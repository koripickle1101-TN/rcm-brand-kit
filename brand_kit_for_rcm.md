# Brand Kit for Tennessee RCM Side Hustle

## Color Palette

- Tennessee Orange: `#FF8200`
- Smokey Gray: `#4B4B4B`
- Black: `#0A0A0A`
- White: `#FFFFFF`

## Brand Rules

### Primary
- Tennessee Orange for buttons, highlights, active states, and important metrics
- Smokey Gray for borders, dividers, muted panels, and secondary structure

### Secondary
- Black for page backgrounds and dark app shells
- White for primary text and high-contrast readability

## GitHub Use

- Use Black for banner backgrounds
- Use Tennessee Orange for titles
- Use White for body text
- Use Smokey Gray for dividers and supporting structure

## Notion Use

- Use a dark cover image with Tennessee Orange accents
- Keep page headers visually consistent
- Use Smokey Gray for supporting visual structure
- Keep text readable with strong contrast

## Vercel UI Use

### CSS Variables

```css
:root {
  --tn-orange: #FF8200;
  --smokey-gray: #4B4B4B;
  --black: #0A0A0A;
  --white: #FFFFFF;
}
```

### Tailwind Theme

```js
theme: {
  extend: {
    colors: {
      brand: {
        orange: '#FF8200',
        gray: '#4B4B4B',
        black: '#0A0A0A',
        white: '#FFFFFF',
      }
    }
  }
}
```
